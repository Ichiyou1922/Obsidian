**注釈リスト**

- **正規直交基底 (Orthonormal Basis)**: お互いに直交し，かつ全ての長さ（ノルム）が $1$ である基底．最も扱いやすい理想的な座標系．
    
- **直交基底 (Orthogonal Basis)**: お互いに直交しているが，長さはバラバラな基底．三角関数系は（正規化しない限り）こちらに分類される．
    
- **射影公式 (Projection Formula)**: ベクトル $\mathbf{v}$ を基底 $\mathbf{u}$ 方向へ分解する際の係数 $c$ は，$\mathbf{u}$ が単位ベクトルでない場合，$c = \frac{\langle \mathbf{v}, \mathbf{u} \rangle}{\|\mathbf{u}\|^2}$ となる．

## 1. 写経セクション (Transcription Section)

### A. 三角関数の「長さ」を測る (Measuring the Norm)

我々はこれまで $\{1, \cos t, \sin t, \cos 2t, \dots\}$ が互いに直交することを確認した．

しかし，これらを「座標軸」として使うには，それぞれの**「目盛りの幅（長さ）」**を知らねばならない．

関数空間におけるベクトル $f(t)$ の「長さの2乗（エネルギー）」は，自分自身との内積で定義される．

$$\|f\|^2 = \langle f, f \rangle = \int_{-\pi}^{\pi} |f(t)|^2 \, dt$$

（※便宜上，区間を対称な $[-\pi, \pi]$ とする．幅は $2\pi$ で同じである）

実際に基底ベクトルの長さを計算する：

1. **交流成分の長さ** ($\cos nt, \sin nt$):
    
    $$\|\cos nt\|^2 = \int_{-\pi}^{\pi} \cos^2(nt) \, dt = \pi$$
    
    計算により，長さは $\sqrt{\pi}$ となる（$1$ ではない！）．
    
2. **直流成分の長さ** ($1$):
    
    $$\|1\|^2 = \int_{-\pi}^{\pi} 1^2 \, dt = [t]_{-\pi}^{\pi} = 2\pi$$
    
    直流成分だけ，長さが $\sqrt{2\pi}$ となり，他の成分より $\sqrt{2}$ 倍「太い」ベクトルである．
    

### B. 係数の「補正」とフーリエ公式の正体 (The Correction Factor)

線形代数において，任意のベクトル $\mathbf{y}$ を，直交基底 $\mathbf{u}$ に射影して係数 $c$ を求める公式は以下の通りである．

$$c = \frac{\langle \mathbf{y}, \mathbf{u} \rangle}{\langle \mathbf{u}, \mathbf{u} \rangle} = \frac{\text{内積}}{\text{基底の長さ}^2}$$

もし基底が長さ1（正規基底）なら分母は1で消えるが，三角関数系は長さ1ではないため，**「長さの2乗」で割って補正**しなければならない．

これこそがフーリエ係数の公式の正体である．

- $a_n$ ($n \ge 1$) の公式:
    
    基底 $\cos nt$ の長さの2乗は $\pi$ である．ゆえに：
    
    $$ a_n = \frac{1}{\pi} \int_{-\pi}^{\pi} f(t) \cos(nt) \, dt$$
    
    （内積を $\pi$ で割っている）
    
- $a_0$ (直流分) の公式:
    
    基底 $1$ の長さの2乗は $2\pi$ である．ゆえに：
    
    $$ a_0 = \frac{1}{2\pi} \int_{-\pi}^{\pi} f(t) \cdot 1 \, dt$$
    
    （内積を $2\pi$ で割っている．これは単なる関数の「平均値」である）
    

**結論**: フーリエ係数につく係数は，積分計算のオマケではなく，**「座標軸の縮尺を揃えるための正規化定数」**である．

## 2. 再構築領域 (Reconstruction Zone)

目を離し，以下の幾何学的構造を手元で再現せよ．

1. **射影の不整合**:
    
    - 長い基底ベクトル $\mathbf{u}$（長さ $>1$）を描く．
        
    - 別のベクトル $\mathbf{v}$ から $\mathbf{u}$ へ垂線を下ろす．
        
    - 単純な内積 $\langle \mathbf{v}, \mathbf{u} \rangle$ は，「影の長さ」$\times$「$\mathbf{u}$ の長さ」になるため，実際の影の長さ（係数）よりも値が大きくなってしまうことを図示せよ．
        
    - 正しい係数を得るには，$\mathbf{u}$ の長さで2回割る（あるいは $\mathbf{u}$ を単位ベクトル化してから内積をとる）必要があることをメモせよ．
        
2. **基底の長さリスト**:
    
    - 基底: $1$ $\rightarrow$ ノルム2乗: $2\pi$
        
    - 基底: $\cos nt, \sin nt$ $\rightarrow$ ノルム2乗: $\pi$
        
    - この「不揃い」が公式の係数の違いを生んでいる構造を書き留めよ．

## 3. 演習領域 (Exercise Zone)

実際に積分計算を行い，基底の「長さ」を自力で導出する．これが係数の分母になることを確認せよ．

課題:

基底ベクトル $u(t) = \cos(t)$ の区間 $[-\pi, \pi]$ におけるノルムの2乗 $\|\cos t\|^2$ を計算する．

**推論プロセス**:

1. 定義に従い積分をセットアップする：
    
    $I = \int_{-\pi}^{\pi} \cos^2 t \, dt$
    
2. 三角関数の半角の公式 $\cos^2 t = \frac{1 + \cos(2t)}{2}$ を適用して次数を下げる：
    
    $I = \int_{-\pi}^{\pi} \frac{1 + \cos(2t)}{2} \, dt$
    
    $I = \frac{1}{2} \int_{-\pi}^{\pi} (1 + \cos(2t)) \, dt$
    
3. 項ごとに積分を実行する：
    
    第1項: $\int_{-\pi}^{\pi} 1 \, dt = [t]_{-\pi}^{\pi} = \pi - (-\pi) = 2\pi$
    
    第2項: $\int_{-\pi}^{\pi} \cos(2t) \, dt = \left[ \frac{1}{2}\sin(2t) \right]_{-\pi}^{\pi} = \_\_\_\_\_\_$
    
    （ヒント: $\sin$ は $2\pi$ 周期であり，端点での値はどうなるか？）
    
4. 結果を統合する：
    
    $I = \frac{1}{2} ( 2\pi + \_\_\_\_\_\_ ) = \_\_\_\_\_\_$
    

結論:

計算結果より，$\cos t$ のノルムの2乗は ______ である．

したがって，信号 $f(t)$ に含まれる $\cos t$ 成分の大きさ（係数 $a_1$）を求める際，内積 $\int f(t)\cos t dt$ をこの値 ______ で割る必要がある．

（事後チェック用ヒント: 第2項の積分値は0になる．最終結果は $\pi$．これが公式の $1/\pi$ の正体である）
