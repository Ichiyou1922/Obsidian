### 10.1 & 10.2 内積の公理的定義

これまで扱ってきたベクトル空間 $V$ には，「遠い・近い」や「垂直」という概念はなかった．

これらを議論するために，$V$ に内積 (Inner Product) という構造を付与する．

定義 (実内積空間):

実ベクトル空間 $V$ において，任意の2つのベクトル $x, y$ に対し実数 $(x, y)$ を対応させる写像が，以下の3条件を満たすとき，これを内積と呼ぶ．

1. 双線形性 (Bilinearity):
    
    線形性が各成分について成り立つ．
    
    $(ax_1 + bx_2, y) = a(x_1, y) + b(x_2, y)$
    
    $(x, ay_1 + by_2) = a(x, y_1) + b(x, y_2)$
    
2. 対称性 (Symmetry):
    
    順序を入れ替えても値は変わらない．
    
    $(x, y) = (y, x)$
    
    （※複素空間の場合は共役対称 $(x, y) = \overline{(y, x)}$ となる．後述）
    
3. 正定値性 (Positive Definiteness):
    
    自分自身との内積は常に正であり，$0$ になるのは自分自身が $0$ の時に限る．
    
    $(x, x) \ge 0$
    
    $(x, x) = 0 \iff x = 0$
    

この内積が定義された空間 $(V, (\cdot, \cdot))$ を**内積空間（計量ベクトル空間）**と呼ぶ．

### 10.3 & 10.4 ノルムと角度の「導出」

内積さえあれば，長さと角度は以下のように**定義（計算）**される．

1. ノルム (Norm / Length):
    
    ベクトルの「長さ」は，自分自身との内積のルートである．
    
    $$\|x\| = \sqrt{(x, x)}$$
    
2. 角度 (Angle):
    
    2つのベクトルの「相関の強さ」を角度とみなす．
    
    $$\cos \theta = \frac{(x, y)}{\|x\| \|y\|}$$
    
    ここで重要なのは，「直交（Orthogonal）」の定義である．
    
    $(x, y) = 0$ のとき，$\theta = 90^\circ$ となり，$x$ と $y$ は直交するという．
    

構造的意義 (The Structure):

「直交」とは，互いの情報（成分）が全く重なっていない状態，すなわち**「無相関」**であることを意味する．

フーリエ級数展開も量子力学の波動関数も，すべてこの「関数同士の直交性（内積がゼロ）」を基盤としている．

### コーシー・シュワルツの不等式の証明

内積空間において，角度 $\cos \theta$ を定義できるのは，以下の不等式が保証されているからに他ならない．

（もし右辺のほうが小さければ，$\cos \theta > 1$ となり定義が破綻する）

定理:

任意の $x, y \in V$ に対して，

$$| (x, y) | \le \|x\| \|y\|$$

等号成立は，$x$ と $y$ が線形従属（平行）のときに限る．

**証明のアルゴリズム（再構築せよ）:**

任意の定数 $t \in \mathbb{R}$ に対して，ベクトル $tx + y$ のノルムの2乗を考える．

正定値性より，$\| tx + y \|^2 \ge 0$ である．

1. 左辺を展開する（双線形性と対称性を使用）：
    
    $\| tx + y \|^2 = (tx + y, tx + y)$
    
    $= t^2(x, x) + 2t(x, y) + (y, y)$
    
    $= \|x\|^2 t^2 + 2(x, y)t + \|y\|^2$
    
2. これは $t$ に関する2次不等式である．
    
    $At^2 + 2Bt + C \ge 0$
    
    ここで $A = \|x\|^2, B = (x, y), C = \|y\|^2$．
    
3. 常に $0$ 以上であるための条件（判別式 $D$）は？
    
    $D/4 = B^2 - AC \le 0$
    
4. これを元の記号に戻すと：
    
    $(x, y)^2 - \|x\|^2 \|y\|^2 \le 0$
    
    $(x, y)^2 \le \|x\|^2 \|y\|^2$
    

平方根をとれば，証明完了である．
