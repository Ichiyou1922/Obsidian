
## 3. 演習領域 (Exercise Zone)

_(Note: 以下の写像が「線形」であるか否かを判定せよ．定義に従って厳密に論証すること)_

問題 1: 線形性の判定

以下の写像 $f: \mathbb{R} \to \mathbb{R}$ について，線形写像であるものには○，そうでないものには×をつけ，その理由（どの公理が破れているか）を記述せよ．

1. $f(x) = 2x$
    
    判定: True
    
    理由: $f(x+y) = 2(x+y) = 2x + 2y = f(x)+f(y)$ かつ $f(cx) = 2cx = c f(x)$．（成立）
    
2. $f(x) = x + 1$
    
    判定: False
    
    理由: 原点の保存 $f(0)=0$ を確認せよ．$f(0) = 1$．ゆえに線形性はない．
    
3. $f(x) = x^2$
    
    判定: False
    
    理由: $f(cx) = (cx)^2 = c^2 x^2$．これは $c f(x)$ と一致するか？ しない．
    

問題 2: 微分作用素の線形性

$V$ を $x$ についての多項式全体の集合とする．写像 $D: V \to V$ を以下のように定義する．

$$D(P(x)) = \frac{d}{dx} P(x)$$

（つまり，多項式を微分する操作である）

この写像 $D$ は線形写像か？

任意の多項式 $P(x), Q(x)$ と定数 $c$ について：

$D(P(x) + Q(x)) = \frac{d}{dx}(P(x) + Q(x)) = \frac{d}{dx}P(x)+\frac{d}{dx}Q(x)$

$D(c P(x)) = \frac{d}{dx}(c P(x)) = c\frac{d}{dx}P(x)$$

結論: 微分とは，関数空間における線形変換である．
