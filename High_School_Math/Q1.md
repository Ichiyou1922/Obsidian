# Diagnostic Exam: Phase 1 (Foundation & Structure)

> Constraint: 計算用紙に手書きで解答を作成し，思考の過程（途中の式変形，図，迷った形跡）を残すこと．
> 
> Scope: 高校数学全範囲 $\cup$ 大学数学基礎（微積・線形・確率・常微分方程式）

## Problem 1: [Linear Algebra $\times$ Complex Plane]

Theme: 演算の同型性と表現行列

複素数 $z = x + iy$ に対して，ある複素数 $\alpha = a + ib$ を掛ける操作 $w = \alpha z$ を考える．

複素数平面上の点 $z$ を 2次元実ベクトル $\vec{v} = \begin{pmatrix} x \\ y \end{pmatrix}$ とみなしたとき，この乗算操作は線形変換として記述できる．

1. $\vec{w} = M_\alpha \vec{v}$ を満たす $2 \times 2$ 行列 $M_\alpha$ を $a, b$ を用いて記述せよ．
    
2. 行列 $M_\alpha$ の固有値 $\lambda$ および固有ベクトルを求めよ．
    
3. この固有値の意味を，複素数 $\alpha$ の極形式 $\alpha = r e^{i\theta}$ との関連から幾何学的に説明せよ．
    

## Problem 2: [Calculus $\times$ Mechanics]

Theme: 減衰振動の構造と臨界点

質量 $m$，抵抗係数 $\gamma$，ばね定数 $k$ を持つ減衰振動系の運動方程式は以下で与えられる．

$$m \frac{d^2x}{dt^2} + \gamma \frac{dx}{dt} + kx = 0 \quad (m, \gamma, k > 0)$$

1. この微分方程式の特性方程式（Characteristic Equation）を記述し，その解を求めよ．
    
2. 系が**「臨界減衰 (Critical Damping)」**となる条件（$\gamma$ の値）を $m, k$ を用いて導出せよ．
    
3. $\gamma$ が臨界値より大きい場合（過減衰）と小さい場合（不足減衰）で，解 $x(t)$ の挙動（振動するか否か）がどう変化するか，特性方程式の解の配置（実数解か複素数解か）に基づいて論理的に説明せよ．
    
    （キミが学習中のLPF/HPFや時定数の概念ともリンクする重要な構造である）
    

## Problem 3: [Probability $\times$ Integration]

Theme: 連続型確率分布の正規化とモーメント

確率変数 $X$ が以下の確率密度関数 (PDF) に従うとする．

$$f(x) = \begin{cases} C x^2 (1 - x) & (0 \le x \le 1) \\ 0 & (\text{otherwise}) \end{cases}$$

1. 全確率が 1 となるための定数 $C$ の値を求めよ．
    
2. 期待値 $E[X]$ および分散 $V[X]$ を導出せよ．
    
3. （記述）なぜ確率密度関数の値 $f(x)$ は 1 を超えてもよいのか？「確率」と「確率密度」の違いを次元（Dimension）の観点から説明せよ．
    

## Problem 4: [Vector Calculus $\times$ Fields]

Theme: 保存場とポテンシャル

2次元ベクトル場 $\vec{F}(x, y) = (2xy + y^3) \vec{i} + (x^2 + 3xy^2) \vec{j}$ を考える．

1. このベクトル場の回転（Rotation / Curl） $\nabla \times \vec{F}$ （2次元ではスカラ量 $\frac{\partial F_y}{\partial x} - \frac{\partial F_x}{\partial y}$）を計算せよ．
    
2. この場が保存力場（Conservative Field）であることを確認し，$\vec{F} = \nabla \phi$ となるスカラーポテンシャル $\phi(x, y)$ を求めよ．（積分定数は無視してよい）
    

## Problem 5: [Sequence $\times$ Limit]

Theme: 非線形漸化式と不動点

数列 $\{a_n\}$ が $a_1 = 1, \quad a_{n+1} = \sqrt{a_n + 2}$ で定義される．

1. この数列が収束すると仮定した場合の極限値 $\alpha = \lim_{n \to \infty} a_n$ を求めよ．
    
2. すべての $n$ において $|a_{n+1} - \alpha| \le k |a_n - \alpha|$ となる定数 $k$ ($0 < k < 1$) が存在することを示し，数列が実際に $\alpha$ に収束することを証明せよ．（平均値の定理を用いてもよい）
    

## Problem 6: [Set $\times$ Topology]

Theme: 開集合の定義と直観

実数直線 $\mathbb{R}$ における通常の距離（ユークリッド距離）を考える．

集合 $A = (0, 1) = \{ x \in \mathbb{R} \mid 0 < x < 1 \}$ が「開集合 (Open Set)」であることを，開集合の定義に基づいて証明せよ．

注: 開集合の定義「任意の点 $x \in A$ に対して，ある $\epsilon > 0$ が存在し，開区間 $(x-\epsilon, x+\epsilon)$ が $A$ に含まれる」を使用すること．

## Problem 7: [Geometry $\times$ Optimization]

Theme: 制約付き最適化と幾何

半径 $R$ の球に内接する直円柱を考える．

1. この直円柱の体積 $V$ を最大にする底面の半径 $r$ と高さ $h$ を求めよ．
    
2. （記述）求めた形状において，球の中心から見た円柱の形状（縦横比など）にどのような対称性や特徴があるか述べよ．
